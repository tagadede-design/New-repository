# 記事タイトル作成ジェネレーター（TitleCraft）

ジャンル・読者ターゲット・記事テーマ・雰囲気を入力すると、note、ブログ、SNS投稿に使える記事タイトル案を10個作成するブラウザツールです。

![Server](https://img.shields.io/badge/Server-不要-44bf8a)

## 特徴

- 入力内容から切り口の異なるタイトル案を10個生成
- 「やさしい」「専門的」「煽り気味」「上品」の4種類の雰囲気に対応
- タイトルを1件ずつ、または10件まとめてコピー可能
- パソコン・タブレット・スマートフォンに対応
- 会員登録、APIキー、サーバー接続は一切不要
- 入力データを外部に送信しないため、プライバシー面でも安心
- HTML / CSS / JavaScriptだけで動作し、買った人がすぐ使える

## ファイル構成

```text
記事タイトル作成ジェネレーター/
├── index.html   # ツール本体
├── style.css    # デザイン・スマホ対応
├── script.js    # タイトル生成・コピー機能
└── README.md    # 使い方・販売用説明
```

## 使い方

1. ダウンロードしたZIPファイルを展開します。
2. フォルダ内の `index.html` をダブルクリックします。
3. ブラウザでツールが開いたら、以下の項目を入力します。
   - ジャンル
   - 読者ターゲット
   - 記事テーマ
   - 雰囲気
4. 「タイトル案を10個つくる」ボタンを押します。
5. 気に入ったタイトルのコピーボタン、または「すべてコピー」を押して利用します。

インストールや会員登録は不要です。Chrome、Edge、Safari、Firefoxなどの一般的なブラウザで利用できます。

### 正しく表示するための確認事項

`index.html`、`style.css`、`script.js` の3ファイルは、ファイル名を変更せず必ず同じフォルダに置いてください。ZIP内の `index.html` を直接開くのではなく、先にZIPを展開してから開きます。表示が古い場合はブラウザを再読み込みしてください。

## 購入者向けの注意事項

- 本ツールが作成したタイトルは、そのままでも編集しても利用できます。
- 生成結果は必ずしも成果、閲覧数、売上などを保証するものではありません。
- 入力内容は外部サーバーに送信・保存されません。
- 本ツール本体の再販売、再配布、複製しての譲渡は禁止です。
- 購入者本人の記事、投稿、仕事、商用コンテンツへのタイトル利用は可能です。

## カスタマイズ

タイトルのパターンを変更したい場合は、`script.js` 内の `titlePatterns` を編集してください。配色や文字サイズは `style.css` 冒頭の `:root` にあるカラー変数を変更すると調整しやすくなっています。
