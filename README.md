# TitleCraft｜記事タイトル作成ジェネレーター

ジャンル・読者ターゲット・記事テーマを入力すると、note、ブログ、SNS投稿などに使える記事タイトル案を10個作成できる、買い切り型のブラウザツールです。

> [!IMPORTANT]
> 購入者向け説明は **`はじめにお読みください.txt`** を確認してください。

## `index.html` だけで動きます

このツールの本体は **`index.html` 1ファイルだけ**です。デザイン用CSSとタイトル生成用JavaScriptは、すべて`index.html`の中に収録されています。

- `style.css`、`script.js`、画像、フォント、外部ライブラリは不要
- インストール、会員登録、APIキー、ローカルサーバーは不要
- インターネット接続がなくても利用可能
- 入力内容や生成結果を外部へ送信しないブラウザ内完結型
- パソコン、タブレット、スマートフォンに対応

## ファイル構成

```text
TitleCraft/
├── index.html                   # デザインと全機能を含むツール本体
├── はじめにお読みください.txt   # 購入者向けの使い方・注意事項
└── README.md                    # GitHub向けの詳細説明・ZIP化手順
```

`index.html`だけでも動作しますが、購入者へ納品する際は`はじめにお読みください.txt`も一緒に同梱してください。READMEはGitHub向けの詳細資料として利用できます。

## 使い方

1. ZIPで受け取った場合は、先にZIPを展開します。
2. 展開したフォルダ内の`index.html`をダブルクリックします。
3. ブラウザで画面が開いたら、次の項目を入力します。
   - ジャンル
   - 読者ターゲット
   - 記事テーマ
   - 雰囲気（やさしい／専門的／インパクト／上品）
4. 「タイトル案を10個つくる」を押します。
5. 表示された各案の「コピー」、または「すべてコピー」を押して利用します。
6. 別の案を見たい場合は「もう一度つくる」を押します。

必須項目が未入力の場合は、画面上に不足項目が表示されます。

> コピー機能は`file://`での利用にも対応するフォールバックを備えています。ブラウザや端末のセキュリティ設定でコピーが許可されない場合は、タイトルを選択して手動でコピーしてください。

## 主な機能

- 入力内容と選択した雰囲気に合わせて10案を生成
- 「やさしい」「専門的」「インパクト」「上品」の4種類に対応
- タイトルを1件ずつコピー
- 10件を番号付きでまとめてコピー
- 未入力項目を分かりやすく案内
- 画面幅に合わせてレイアウトが変わるレスポンシブデザイン
- サンプル入力の自動セット
- 同じ入力内容での再生成

## 推奨環境と確認方法

Chrome、Edge、Safari、Firefoxなど、JavaScriptが有効な一般的な最新ブラウザで利用してください。

確認するときは、`index.html`をダブルクリックして開き、以下をチェックします。

1. 色、カード、入力欄などのデザインが表示される
2. 必須項目を入力するとタイトルが10個表示される
3. 各「コピー」と「すべてコピー」が反応する
4. インターネット接続を切っても画面と生成機能が動く
5. `index.html`以外のツール用ファイルを置かなくても動く

4. 「送る」→「圧縮（zip形式）フォルダー」を選びます。
5. 作成されたZIPを一度展開し、展開先の`index.html`が動くことを確認してから納品します。
