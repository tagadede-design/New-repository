# PostPalette｜AI発信テンプレジェネレーター

個人事業主・副業者が商品やサービスを発信するときに使える、**note・X・Instagram対応のAIプロンプト生成ツール**です。商品情報を一度入力すると、媒体と目的に合わせた10種類のプロンプトをブラウザ内で生成します。

## 特徴

- HTML / CSS / JavaScriptだけで動作
- 外部API、外部ライブラリ、会員登録、APIキーは不要
- 入力内容を外部へ送信しないローカル完結型
- PC・タブレット・スマートフォン対応
- サンプル入力、媒体タブ、個別コピー、一括コピーを搭載
- 媒体別の定義をJavaScriptの`platforms`オブジェクトに集約し、特化版へ拡張しやすい構成

## 収録プロンプト

### note

1. note記事タイトル案
2. note記事構成
3. 販売導線付きnote本文

### X

1. X短文投稿20本
2. Xスレッド投稿
3. 無料相談や購入につなげる告知投稿

### Instagram

1. Instagramキャプション
2. カルーセル投稿構成
3. リール台本
4. ストーリーズ案

## 使い方

1. ZIPで受け取った場合は、先にZIPを展開します。
2. 展開したフォルダの`index.html`をChrome、Edge、Safari、Firefoxなどのブラウザで開きます。
3. 商品名、ターゲット、商品の強みなどを入力します。
4. 「AIプロンプトを作成する」を押します。
5. note・X・Instagramのタブから使いたい媒体を選びます。
6. 必要なプロンプトの「コピー」を押します。
7. ChatGPTなど任意の生成AIの入力欄へ貼り付け、そのまま送信します。
8. AIから返された文章の事実、価格、リンク、表現を確認・編集してから投稿します。

入力例を確認したい場合は「入力例を自動セットして試す」を押してください。

> このツールはAIへ接続して投稿本文を生成するものではありません。入力内容を整理し、任意の生成AIへ渡すためのプロンプトを作成します。

### うまく使うコツ

- 「ターゲット」は「30代女性」だけでなく、現在の状況や望んでいることまで書く
- 「悩み」は本人が普段使いそうな言葉で具体的に書く
- 「商品の強み」は競合との違い、サポート内容、取り組みやすさを書く
- 「ベネフィット」は購入後にどんな状態へ近づけるかを書く
- AIの出力が合わない場合は、生成されたプロンプト末尾へ「もっと○○な表現にしてください」などの要望を足す

## 入力項目

- 商品名・サービス名（必須）
- 商品ジャンル（必須）
- ターゲット（必須）
- ターゲットの悩み
- 商品の強み
- ベネフィット
- 販売価格
- 投稿の雰囲気
- 誘導したい行動
- 申込方法・連絡方法
- 避けたい表現

## ファイル構成

```text
.
├── index.html                  # CSS・JavaScriptを含むツール本体
├── README.md                   # 開発者・利用者向け説明
└── はじめにお読みください.txt  # 配布時の案内
```

## 起動方法

インストールは不要です。`index.html`を直接ダブルクリックしてください。ローカルサーバーを使う場合は、たとえば次のコマンドでも起動できます。

```bash
python3 -m http.server 8000
```

その後、ブラウザで `http://localhost:8000` を開きます。

## カスタマイズ

媒体ごとのテンプレートは`index.html`内の`platforms`オブジェクトで管理しています。媒体特化版を作る場合は、不要な媒体を削除し、対応するタブを調整してください。新しい投稿形式は、対象媒体の配列に次の形式で追加できます。

```js
{
  title: "画面に表示する名称",
  task: "生成AIへ渡す具体的な依頼内容"
}
```

共通の商品情報と安全ルールは`makePrompt()`で組み立てています。

## 動作確認

- 必須項目が空の場合にブラウザ標準の入力案内が表示される
- サンプル入力が全項目へ反映される
- 3媒体、合計10種類のプロンプトが表示される
- タブを切り替えると媒体別の結果が表示される
- 個別コピーと表示中の全件コピーが動作する
- 画面幅に応じて1カラムへ切り替わる
- 外部通信なしで動作する

## 対応環境

JavaScriptが有効な最新のChrome、Edge、Safari、Firefoxを推奨します。ブラウザのセキュリティ設定によりClipboard APIが利用できない場合に備え、コピー処理にはフォールバックを用意しています。

## 注意事項

- 入力内容と生成したプロンプトは保存されません。必要な内容は画面を閉じる前にコピーしてください。
- 生成AIが出力した投稿は、公開前に事実関係、各SNSの規約、広告・景品表示関連のルールをご自身で確認してください。
- 本ツールおよび生成プロンプトは、閲覧数や売上などの成果を保証するものではありません。

## 納品方法

販売商品として購入者へ渡す場合は、次の手順を推奨します。

1. `index.html`、`README.md`、`はじめにお読みください.txt`を1つのフォルダへ入れます。
2. フォルダ名を商品名やバージョンがわかる名称にします（例：`PostPalette_v1.0`）。
3. フォルダをZIP形式で圧縮します。
4. 作成したZIPを別の場所へ一度展開します。
5. 展開先の`index.html`を開き、サンプル入力、生成、3媒体のタブ切り替え、コピーを最終確認します。
6. 販売プラットフォームのファイル納品機能、購入後メッセージ、またはダウンロードURLで購入者へ渡します。

### Windowsでの圧縮

フォルダを右クリックし、**「送る」→「圧縮（zip形式）フォルダー」**を選択します。

### macOSでの圧縮

Finderでフォルダを右クリックし、**「“フォルダ名”を圧縮」**を選択します。

### コマンドでの圧縮

```bash
zip -r PostPalette_v1.0.zip index.html README.md はじめにお読みください.txt
```

> ZIPの中に入ったまま`index.html`を開くと、環境によって正常に使えない場合があります。購入者には「最初にZIPを展開する」ことを必ず案内してください。
