# TitleCraft｜記事タイトル作成ジェネレーター

ジャンル・読者ターゲット・記事テーマを入力すると、note、ブログ、SNS投稿に使える記事タイトル案を10個作成する買い切り型のブラウザツールです。

## 特徴

- 入力内容から切り口の異なるタイトル案を10個生成
- 「やさしい」「専門的」「インパクト」「上品」の4種類の雰囲気に対応
- タイトルを1件ずつ、または10件まとめてコピー可能
- パソコン・タブレット・スマートフォンに対応
- 会員登録、APIキー、インストール、サーバー接続は不要
- 入力データを外部に送信せず、すべてブラウザ内で処理
- 外部画像・外部フォント・外部ライブラリを使用しない完全ローカル仕様

## ファイル構成

```text
TitleCraft/
├── index.html   # ツール本体
├── style.css    # デザイン・レスポンシブ対応
├── script.js    # タイトル生成・コピー機能
└── README.md    # 使い方・商品説明
```

4ファイルは同じフォルダに置いてください。`index.html` は同じ階層の `./style.css` と `./script.js` を読み込みます。

## 使い方

1. 購入・ダウンロードしたZIPファイルを展開します。
2. 展開したフォルダ内の `index.html` をダブルクリックします。
3. ブラウザでツールが開いたら、次の項目を入力します。
   - ジャンル
   - 読者ターゲット
   - 記事テーマ
   - タイトルの雰囲気
4. 「タイトル案を10個つくる」ボタンを押します。
5. 各案の「コピー」ボタン、または「すべてコピー」を押して利用します。

インターネット接続やローカルサーバーは不要です。Chrome、Edge、Safari、Firefoxなどの一般的なブラウザで、`file://` のまま動作します。

## 動作確認のポイント

- CSSが反映されない場合は、`index.html`、`style.css`、`script.js`が同じフォルダにあるか確認してください。
- ZIPファイルの中身を直接開かず、先にすべて展開してから使用してください。
- コピー機能がブラウザの制限を受ける場合は、表示されたタイトルを選択して手動でコピーしてください。
- スマートフォンでは、生成後に結果欄まで自動的にスクロールします。

## 購入者向けの注意事項

- 作成したタイトルは、そのままでも編集しても利用できます。
- 生成結果は、閲覧数や売上などの成果を保証するものではありません。
- 入力内容は外部サーバーへ送信・保存されません。
- ツール本体の再販売、再配布、複製しての譲渡は禁止です。
- 購入者本人の記事、投稿、仕事、商用コンテンツへのタイトル利用は可能です。

## カスタマイズ

タイトルのパターンは `script.js` 内の `titlePatterns` で変更できます。配色は `style.css` 冒頭の `:root` にある変数で変更できます。
